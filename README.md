# xcel

Java libraries for validating Excel-related workflows.

## Modules
- `xcel-validation-core` – core validation APIs and rules
- `xcel-validation-poi` – Apache POI-based implementations

## Requirements
- Java 17+
- Gradle (wrapper included)

## Build
```
./gradlew build
```

## Tests
```
./gradlew test
```
